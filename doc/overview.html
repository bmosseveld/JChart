The JChart Library can be used to visualize different types of charts in a Java application.

It currently supports line charts, bar charts, pie charts and scatter charts.
These charts can be easily integrated in a Java application because they are extensions of JPanel.

The main class is JChart with extensions:

  • JBarChart
  • JLineChart
  • JPieChart
  • JScatterChart
  • JBoxChart
